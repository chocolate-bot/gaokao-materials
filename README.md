# 高考作文素材库

> 来自《人民日报》的每日精选，专为高考作文准备

## 📚 项目介绍

每天自动整理《人民日报》中的高考作文素材，包括：

- **原文概括**：200字以内的核心内容提炼
- **核心优势**：分析为什么适合高考
- **金句储备**：3-5个可直接引用的名句
- **适用话题**：5-6个高考作文话题
- **结构方法**：论证结构分析
- **应用建议**：如何使用这些素材

## ✨ 特点

- ✅ **每日更新**：每天早上8点自动整理
- ✅ **精选提炼**：只选适合高考的论证性文章
- ✅ **Orange UI**：采用Neo-Brutalist设计风格
- ✅ **响应式设计**：适配手机/iPad/电脑
- ✅ **无障碍友好**：符合WCAG AAA标准

## 🚀 如何使用

### 日常积累（每天10分钟）
1. 浏览金句储备，挑选1-2句背下来
2. 看适用话题，思考这个话题你会怎么写
3. 读原文概括，快速获取核心观点

### 专题训练（每周1次）
1. 选一个话题，比如"阅读与成长"
2. 找相关素材，从积累的素材中挑选
3. 模仿论证结构，比如用并列式写三个分论点
4. 引用金句，在恰当位置引用《人民日报》的金句

### 考前冲刺（考前1个月）
1. 按话题整理素材，把同类话题的素材放在一起
2. 准备万能论据，比如"航天精神""文化传承"等
3. 背诵核心金句至少20-30句

## 🎨 设计系统

采用 **Orange UI** 设计系统：

- **背景**：#F7F3EC（米白）
- **主色**：#7BCBFF（天蓝）
- **强调色**：#FFE66A（柔黄）
- **成功色**：#6FD8C7（薄荷）
- **Neo-Brutalist风格**：硬阴影、清晰边框

## 🤖 自动化

使用 Hermes Agent 的 cron job 功能实现：

- **执行时间**：每天早上8点
- **数据源**：《人民日报》电子版
- **输出格式**：MD文件 + HTML文件
- **存储位置**：Obsidian vault + GitHub
